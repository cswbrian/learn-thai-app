# Sawadee

A modern web application for learning Thai language, featuring interactive lessons on consonants, vowels, numbers, and quizzes.

## Description

Sawadee helps you learn Thai through interactive flashcards and quizzes. Explore Thai consonants (initial and final), vowels, and numbers with an intuitive interface designed for effective learning.

## Getting Started

### Installation

```bash
npm install
# or
pnpm install
```

### Development

Start the development server:

```bash
npm run dev
# or
pnpm dev
```

### Build

Build for production:

```bash
npm run build
# or
pnpm build
```

### Preview

Preview the production build:

```bash
npm run preview
# or
pnpm preview
```

### Lint

Run ESLint:

```bash
npm run lint
# or
pnpm lint
```
